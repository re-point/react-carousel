## RTL layout

Carousel supports the right to left layout.

```jsx render
// import Carousel from '@re_point/react-carousel';
// import '@re_point/react-carousel/lib/style.css';

<Carousel plugins={['rtl', 'arrows']}>
  <img src={imageOne} />
  <img src={imageTwo} />
  <img src={imageThree} />
</Carousel>
```
