## Default Arrows

You can turn default arrows on, using the `arrows` plugin.

```jsx render
// import Carousel from '@re_point/react-carousel';
// import '@re_point/react-carousel/lib/style.css';

<Carousel plugins={['arrows']}>
  <img src={imageOne} />
  <img src={imageTwo} />
  <img src={imageThree} />
</Carousel>
```
