## Simple slide

For simple usage, just pass slides as children. You don't need any configuration!

```jsx render
// import Carousel from '@re_point/react-carousel';
// import '@re_point/react-carousel/lib/style.css';

<Carousel>
  <img src={imageOne} />
  <img src={imageTwo} />
  <img src={imageThree} />
</Carousel>
```
